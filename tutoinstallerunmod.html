Tutoriel : Installation de Mod

Tuto crée par MultiGame
1
Téléchargez le mod en .zip sur notre site web.
2
Placez le fichier dans : Documents / My Games / FarmingSimulator2022 / mods
3
Lancez votre jeu et assurez-vous de cocher les mods lors de la sélection de votre partie.
ATTENTION : Ne décompressez JAMAIS les fichiers .zip. Le serveur ne reconnaîtra pas les fichiers s'ils sont extraits.
Un problème ? Suis bien la vidéo ou contacte un admin sur Discord.
